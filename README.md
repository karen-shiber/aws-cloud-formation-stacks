# Implementando Stack com AWS CloudFormation (DIO)

Este repositório foi desenvolvido para o desafio da (DIO) chamado “Implementando sua Primeira Stack com AWS CloudFormation”.

Objetivos do Projeto

Aplicar os conceitos de IaC (Infrastructure as Code).

Criar e estruturar um template CloudFormation no formato YAML.

Simular a implantação de um recurso simples (bucket S3).

Documentar os passos e aprendizados de forma clara e didática.

Publicar o projeto no GitHub como portfólio técnico.
